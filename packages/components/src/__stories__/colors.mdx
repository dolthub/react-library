# Colors

Colors we use in our design system.

## Base Colors

<ColorPalette>
  {Object.entries(staticColors)
    .map(([colorName, colorObj], i) => (
      <ColorItem key={i} title={colorName} colors={Object.values(colorObj)} />
    ))}
</ColorPalette>

## Theme Colors

Colors with <code>(c)</code> share a name among all themes but have different values.

### DoltHub

<ColorPalette>
  {Object.entries(dolthubTailwindColorTheme).map(([k, v], i) => {
    return (
      <ColorItem key={i} title={`${k.replace("rgb-", "")} (c)`} colors={[`rgba(${v})`]} />
    )
  })}
</ColorPalette>
