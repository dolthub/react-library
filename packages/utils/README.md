# @dolthub/web-utils

A library of useful utilities for the web.

## Installation

```
% yarn add @dolthub/web-utils
```

or

```
% npm install @dolthub/web-utils
```

### Peer dependencies

The following peer dependency may need to be installed in your project:

| Package           | Required                             |
| ----------------- | ------------------------------------ |
| `node-sql-parser` | No (only needed if using sql parser) |

```
% yarn add node-sql-parser
```
